    Файлы в данной папке предназначены для настройки распределения вероятности происшествия
в зависимости от состояния службы и генерации веерных инцидентов:
      - input.csv — содержит по три ключевые точки на каждую тяжесть происшествия. Первый столбец —
тяжесть, второй — состояние службы в процентах (х), третий — раз в сколько лет случается ЧС (у).
      - probabilities.txt — input.csv в понятном для восприятия виде.
      - Emergency_compiler.py — из input.csv и папки texts генерирует файлы для работы EmergencyService.cpp
      - e_ids.txt — промежуточный файл для веерных инцидентов
      - show.m — выводит соответствующие графики вероятности для каждой тяжести и общей вероятности ЧС.
      - coefficients.txt — коэффициенты a, b, c: "y = a * exp(-b * x^2) + c", в порядке abcabcabcabc... начиная
с самой низкой тяжести.
